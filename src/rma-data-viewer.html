Use this tool to retrieve RMA data in text file format (UTF-8 encoded) for the state or county selected on the map and the years, commodities, and/or causes of loss specified in the Now Viewing panel. All data are provided courtesy of the USDA Risk Management Agency and the US Census Bureau. Details

  1. To download data for a particular state or county, first select the state or county from the map. Data for all states or counties will be retrieved if no map unit is selected; however, month, commodity, and cause of loss columns can only be included if a state or county has been specified. To retrieve data for all counties in a particular state, select the state from the map and select County as a data column.
  2. To download data for particular years, commodities, and/or causes of loss, first specify these parameters in the Now Viewing panel. The Now Viewing panel can also be used to specify the measurement unit used to report payment indemnity, acreage, count, liability, and/or subsidy.
  3. To download the full data set, click here: gz (31MB), zip (35MB), uncompressed (313MB)
